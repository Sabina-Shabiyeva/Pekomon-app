# Pekomon Pokedex Game

A dynamic, high-performance React application built with Vite that fetches data from the PokeAPI to simulate a "battle" between two randomized teams. 

## Features
- **Dynamic Data Fetching:** Integrates with PokeAPI to retrieve real-time Pokemon statistics.
- **Randomized Gameplay:** Every page refresh redistributes Pokemon into two competing teams.
- **Winner Determination:** Automatically calculates the total Experience (EXP) for each team to declare a winner.
- **Fully Responsive:** Modern UI built with Flexbox and CSS Modules, optimized for mobile, tablet, and desktop screens.

## Technical Architecture
- **React + Vite:** Leveraging Vite for lightning-fast Hot Module Replacement (HMR) and optimized build performance.
- **Props-Driven Design:** The application follows a stateless architecture, utilizing `props` and component composition for clean, predictable data flow.
- **Type Safety:** Implements `PropTypes` for robust runtime type checking of component data.
- **Modular Styling:** Uses **CSS Modules** to ensure scoped styles and prevent global CSS conflicts.
